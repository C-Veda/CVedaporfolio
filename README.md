## üîÑ Project Workflow

1. **Planning Phase**
   - Decided the sections to be included in the website such as Profile, Skills, Education, Projects, Achievements, Certifications, Resume, and Contact.
   - Created a simple wireframe to plan the layout of each section.

2. **Design Phase**
   - Designed the layout using HTML5 structure tags.
   - Applied styling using CSS3 and Tailwind CSS for a modern dark-themed responsive design.
   - Added icons and visual elements using Font Awesome.

3. **Development Phase**
   - Implemented smooth scrolling and navigation using JavaScript.
   - Created dynamic sections like skill bars, project cards, and certification grid.
   - Developed the contact form using HTML and connected it to PHP for processing.

4. **Database Integration**
   - Created a MySQL database to store contact form entries such as name, email, and message.
   - Connected PHP with MySQL to insert and retrieve data.

5. **XML & XSLT Integration**
   - Stored academic/project data in an XML file.
   - Used XSLT to transform the XML data into an HTML table view on the website.

6. **Testing Phase**
   - Tested the website on different browsers like Chrome and Edge.
   - Verified responsiveness on desktop and mobile devices.
   - Checked form validation and database storage functionality.

7. **Deployment Phase**
   - Uploaded all files to GitHub repository.
   - Enabled GitHub Pages to make the website live and accessible online.
